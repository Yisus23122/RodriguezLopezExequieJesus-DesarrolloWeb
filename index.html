Objetivos

Al finalizar el curzo se espera que las y los estudiante puedan:

  • Crear páginas web utilizando HTML,CSS,SVG y JavaScript
  • Escribir 100% del código de un sitio web
  • Diseñando una web que se adapte a diferentes tamaño de pantalla y gestionar la navegación
  • Crar elementos animados que interactúen con el usuario
  • Consumir APIs con JavaScript para colocar contenido en el sitio
